# Mecha-15 Romi Project

Documentation and resources for the Mecha-15 Romi project, a term project for ME 405 — Mechatronics at Cal Poly. This project involves building an autonomous line-following robot using the Pololu Romi chassis, STM32 Nucleo microcontroller, and MicroPython programming language.
